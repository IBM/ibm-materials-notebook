# Analytical Balance

  • Description: Measurements using an analytical balance
  • Member Of:
    • Sample
  • Reference Group Properties:
    • Yield
    • Mass
  • Properties: None
  • Example:
    sample Exp-I-1A {

        analytical_balance Exp-I-1A-mass {
            @Product_A {
                yield: 60%;
                mass: 600 mg;
            };
        };
    }

# Chemical

  • Description: Describes a small-molecule chemical reference for use in an experiment.

  • Member Of: None

  • Reference Group Properties: None

  • Properties:

    • Molecular Weight
    • Density
    • State
    • SMILES
    • InChi
    • InChi Key

  • Example:

    chemical lLactide {
    molecular_weight: 144.12 g/mol;
    state: "solid";
    smiles: "C[C@H]1C(=O)O[C@H](C(=O)O1)C";
}

# Component

  • Description: Describes a physical component in a continuous flow reactor.

  • Member Of:

    • Reactor Graph
    • Reactor

  • Reference Group Properties: None

  • Properties:

    • Length
    • Volume
    • Inner Diameter
    • Outer Diameter

  • Example:

    Note: The following illustrate valid arrangements of the component group. For more detailed description of a reactor_graph setup see the flow reaction tutorial.

    reactor_graph FlowReactorA {
    reactor PolymerizationReactor {
        component Mixer {
            description: "PEEK T-Mixer (0.05 cm ID)";
            inner_diameter: 0.05 cm;
        };

        component Reactor_Tubing {
            description: "PFA Reactor Tubing (14 cm, 0.05 cm ID)";
            inner_diameter: 0.05 cm;
            length: 14 cm;
            volume: 0.02749 ml;
        };
    };
}

# Container

  • Description: Describes a container group in a polymer_graph representation. Contains references to edges, nodes, and other container groups for describing the polymer structure

  • Member Of:

    • Polymer Graph
    • Container

  • Reference Group Properties: None

  • Properties:

    • Nodes
    • Connections

  • Example:

    polymer_graph mPEG_Graph {
    nodes: [ @eg_MeO ];
    <@eg_MeO.R => @PEG_Block.p_PEO.R>;

    container PEG_Block {
        nodes: [ @p_PEO ]
        <@p_PEO.Q => @p_PEO.R>;
    };
}




     
     
     
     

# Differential Scanning Calorimetry

  • Description: Differential scanning calorimetry to measure various thermal properties of a small-molecule chemical or polymeric material.

  • Member Of:

    • Sample

  • Reference Group Properties:

    • Crystalization Temperature
    • Melting Temperature
    • Glass Transition Temperature

  • Properties:

    • File
    • Heating Rate

  • Example:

    Note: file property is still under active development.

        sample Exp-I-1A {

        dsc Exp-I-1A-dsc {
            file: @DSC_Trace_A;
            heating_rate: 10 degC/min;

            @Polymer_Product_A {
                temp_melt: 150 degC;
                temp_glass: 60 degC;
            };
        };
    }

# Fragment

  • Description: A fragment is a discrete chemical entity, such as an end group or repeate unit, that may be referenced one or more times within a polymer graph representation.

  • Member Of: None

  • Properties:

    • SMILES
    • Molecular Weight

  • Example:

        fragment p_PEO {
        molecular_weight: 44.05 g/mol;
        smiles: "[Q]OCC[R]";

        point R {
            quantity: 1;
        };

        point Q {
            quantity: 1;
        };
    }

# Reaction

  • Description: Defines a batch reaction for an experimental record.

  • Member Of: None

  • Reference Group Properties:

    • Mass
    • Volume
    • Moles
    • Pressure
    • Roles

  • Properties:

    • Temperature
    • Volume
    • Reaction Time

  • Example:

    reaction BatchRxn {
    temperature: 22 degC;
    reaction_time: 10 s;

    @mPEG-OH {
        mass: 5 mg;
        roles: [ "initiator" ];
        limiting: true;
    };

    @kOtBu {
        mass: 5 mg;
        roles: [ "catalyst", "reagent" ];
    };

    @lLactide {
        mass: 1440 mg;
        roles: [ "monomer" ];
    };

    @THF {
        volume: 2 ml;
        roles: [ "solvent" ];
    };

    @mPEG-PLLA {
        roles: [ "product" ];
    };
}

# Reactor

  • Description: Defines a container for component elements that belong to a single reactor element within a reactor graph. Treated as a single volume for computing residence times and stoichiometry.

  • Member Of:

    • Reactor Graph

  • Reference Group Properties: None

  • Properties: None

  • Example:

    Note: The following example would be nested inside a reactor graph

    reactor PolymerizationReactor {
    component Mixer {
        description: "PEEK T-Mixer (0.05 cm ID)";
        inner_diameter: 0.05 cm;
    };

    component Reactor_Tubing {
        description: "PFA Reactor Tubing (14 cm, 0.05 cm ID)";
        inner_diameter: 0.05 cm;
        length: 14 cm;
        volume: 0.02749 ml;
    };
};

    • Date

      sample Exp-I-1A {

# Solution

  • Description: Describes a stock solution of reagents for use in a flow reaction.

  • Member Of: None

  • Reference Group Properties:

    • Mass
    • Volume
    • Moles
    • Pressure
    • Roles

  • Properties: None

  • Example:

      solution Catalyst_Solution {
      @KOMe {
          mass: 144 mg;
          roles: [ "catalyst" ];
      };

      @UreaCatalyst {
          mass: 500 mg;
          roles: [ "catalyst" ];
      };

      @THF {
          volume: 8 ml;
          roles: [ "solvent" ];
      };
  }
